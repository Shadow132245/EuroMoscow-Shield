Developer API

Integrate the powerful EuroMoscow Shield V25 engine directly into your applications. Automate obfuscation for Python, JavaScript, Lua, PHP, and HTML.

Endpoint

POST {{ request.url_root }}process

Request Parameters (JSON)

Parameter Type Description
code String The source code you want to process.
action String Either 'encrypt' or 'decrypt'.
lang String 'python', 'javascript', 'lua', 'php', 'html'.
options Array List of layers (e.g., ['rename', 'marshal', 'xor']).

Python Example

import requests

url = "{{ request.url_root }}process"

payload = {
    "code": "print('Hello World')",
    "action": "encrypt",
    "lang": "python",
    "options": ["rename", "marshal", "xor"]
}

response = requests.post(url, json=payload)
print(response.json()['result'])
