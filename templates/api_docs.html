POST /process

Send Python code to be obfuscated or decrypted using a JSON payload.

POST {{ request.url_root }}process

Request Parameters

Key Type Description
code String The Python source code.
action String 'encrypt' or 'decrypt'
options List ['rename', 'marshal', 'zlib', 'rot13', 'xor', 'base64']

Python Example

import requests

# Auto-detects your current host
url = "{{ request.url_root }}process"

payload = {
    "code": "print('EuroMoscow System')",
    "action": "encrypt",
    "options": ["rename", "marshal", "xor"]
}

response = requests.post(url, json=payload)
print(response.json()['result'])
