API Reference

Integrate PyShield's powerful obfuscation engine directly into your own applications. Our REST API allows you to obfuscate or decrypt Python code programmatically.

Process Code

This is the main endpoint to handle both encryption and decryption.

POST /process

Request Parameters (JSON)

Parameter Type Description
code string The Python source code to process.
action string Either 'encrypt' or 'decrypt'.
options array List of layers: ['rename', 'zlib', 'base64', 'hex'].

Example Usage

Here is how you can use the API using Python requests library.

import requests

url = "http://127.0.0.1:5000/process"

payload = {
    "code": "print('Hello World')",
    "action": "encrypt",
    "options": ["rename", "zlib", "base64"]
}

response = requests.post(url, json=payload)
print(response.json()['result'])
