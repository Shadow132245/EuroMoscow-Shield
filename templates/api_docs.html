Developer API Reference

Automate your code protection workflow using the V13 Engine. Supports Python, JavaScript, Lua, PHP, and HTML.

Endpoint

POST {{ request.url_root }}process

Request Body (JSON)

ParameterTypeDescription
codeStringSource code to obfuscate.
langString'python', 'javascript', 'lua', 'php', 'html'
levelString'fast', 'balanced', 'max'

Example Usage (Python)

import requests

url = "{{ request.url_root }}process"

payload = {
    "code": "print('Hello V13')",
    "lang": "python",
    "level": "max"
}

res = requests.post(url, json=payload)
print(res.json()['result'])

Example Usage (Node.js)

const axios = require('axios');

axios.post("{{ request.url_root }}process", {
    code: "console.log('Secure')",
    lang: "javascript",
    level: "balanced"
}).then(res => console.log(res.data.result));
