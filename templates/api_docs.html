POST /process

Send code to be obfuscated or decrypted using a JSON payload.

POST {{ request.url_root }}process

Request Parameters

KeyTypeDescription
codeStringThe source code.
actionString'encrypt' or 'decrypt'
langString'python', 'javascript', 'lua'
optionsList['rename', 'marshal', 'zlib', 'rot13', 'xor', 'base64', 'hex']

Example

import requests
url = "{{ request.url_root }}process"
payload = { "code": "print('Hello')", "action": "encrypt", "lang": "python", "options": ["rename"] }
res = requests.post(url, json=payload)
print(res.json()['result'])
