POST /process

Send JSON payload to encrypt or decrypt code.

Python Example

import requests
payload = {
    "code": "print('Hello')",
    "action": "encrypt",
    "lang": "python",
    "options": ["rename", "marshal"]
}
res = requests.post("https://euro-moscow-shield.vercel.app/process", json=payload)
print(res.json()['result'])
